# Apache SystemDS (apache-systemds)
Apache SystemDS is an open-source ML system for the end-to-end data science lifecycle from data integration to model training and deployment. It provides a declarative machine learning language (DML) and automatic optimization.

## APIs

### Apache SystemDS
SystemDS provides the DML (Declarative Machine Learning) scripting language, a Python API for ML pipeline development, and a Java API for executing ML algorithms with automatic optimization for distributed execution.
